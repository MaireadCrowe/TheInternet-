Range of Purposes

There are many uses for the internet including communication, information search, file sharing/downloading and e Commerce. We will outline each of these.

Communication:

Voice Communication: Audio conferencing, video conferencing and VoIP.
Audio conferencing is a connection between two computers. The participants need access to a microphone and speakers in order to communicate. Examples would be Skype or AOL.

Video/Wed Conferencing is similar to audio conferencing. The main difference between the two is that users are able to see each other.  A webcam is required to use this service. Zoom and Teams are examples of video conferencing.

 VoIP, stands for Voice over Internet protocol. It is a phone service through an internet connected. Users of this are required to have an adapter and a unique phone number. The user can send and receive calls using a computer. Examples of VoIP are WhatsApp and Viber.
Written Communication: Email, Instant Messaging, IRC, SMS and Wireless Communications.
Email is the transmission of messages over communications networks. In order to communicate with others, you require the recipients email address. Gmail and Outlook are examples of companies that host your email. Email is a very quick and effective way to communicate as it only takes seconds to arrive at its destination. A downside of email is spam and the potential viruses that it could contain.

Instant messaging is a text-based chat. It allows users to chat privately with another user over the Internet in real time. This is beneficial when you are looking to contact someone quickly as you can see if the person you want to speak with is online and a notification will appear on their screen immediately.

IRC stand for Internet Relay Chat. It is a multi-user chat system which allows people to talk in “rooms” privately.

SMS and Wireless Communications SMS is short for short message service. It is a global wireless service that allows transmission of messages between mobile phone users.

Information Search:

“Google it” is a common phrase we hear every day. It has the answers to all our questions. We use a search engine which is a program that searches for specific keywords and returns a list of documents found where the keywords were found. The more detailed the search request, the more accurate results are found. A search engine used an algorithm to create its directory so that ideally only correct results are returned.

File Sharing/Downloading:

File Sharing:

This is the public or private sharing of computer data or space in a network. There are various levels of access of the data. File sharing allows multiple people to use the same file. The user can read, modify, copy or print it, again depending on the level of access granted by the administrator.
File Downloading:
Peer to peer file sharing. This is when users can use software that connects to a peer to peer network to search for shared files on the computers of other users who are also connected to the network. This type of sharing was heavily used for sharing music and films. Popular networks are Limewire and PirateBay.

File hosting services are a simplified version of peer to peer software. They can be used with Internet collaboration tools like email, forums or blogs. These sites host files so that others can download them.

File sync and sharing allows the user to access files on a network such as Google Drive or OneDrive. These files can be updated and the user will have instant access to the live update using the cloud-based service. These files can be accessed or shared easily through a website or mobile app.

ECommerce:

Ecommerce is the buying and selling of goods and services over the internet. There are four main types of E Business models.

  1. Business to Business
  2. Consumer to Consumer
  3. Consumer to Business
  4. Business to Consumer

The term ecommerce also encompasses other activities including online auctions, internet banking, payment gateways, and online ticketing. (Anon., 2022)
